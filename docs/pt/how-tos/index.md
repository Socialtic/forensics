# Guias de instruções

Nesta seção, você encontrará materiais de guias passo a passo. De acordo com [a estrutura Diataxis] (https://diataxis.fr), **o objetivo dos guias passo a passo é ser capaz de executar uma tarefa** por meio de instruções passo a passo. Ele pressupõe que quem segue o guia já tem as competências necessárias para executar as tarefas, portanto, concentra-se nas etapas necessárias para chegar a um resultado, corretamente e de acordo com as práticas recomendadas.

Por exemplo, uma receita culinária ensina ao leitor como realizar uma tarefa, nesse caso, preparar um prato. Ela pressupõe que a pessoa já tenha conhecimentos básicos, como cortar alimentos, misturar com um mixer etc.
