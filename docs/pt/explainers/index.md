# Explicadores

Nesta seção, você encontrará materiais explicativos ou explicadores. De acordo com [a estrutura Diataxis] (https://diataxis.fr), os explicadores são materiais para **aprender aspectos teóricos**, que fornecem contexto sobre um tópico específico. Eles permitem que os conceitos sejam compreendidos como parte de tópicos mais amplos e geralmente ajudam a esclarecer o "porquê".
