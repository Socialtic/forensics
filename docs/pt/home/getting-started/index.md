# Primeiras etapas

Nesta seção, você encontrará informações básicas sobre este repositório, incluindo uma **descrição do público-alvo**, detalhes sobre como o conteúdo está organizado usando **ditaxis** e sugestões para **navegar** pelos materiais.  

## O que é este repositório de documentação?

Este é um repositório de **documentação técnica** que visa estabelecer uma base de conhecimento testada, flexível e acessível para **promover a análise forense consensual em benefício da sociedade civil**. Para organizar o conteúdo, é usada a estrutura de referência de documentação técnica [Diataxis] (https://diataxis.fr/).

A análise forense digital permite **descobrir e examinar dados de dispositivos e mídias eletrônicas** com o objetivo de identificar, recuperar, documentar e interpretar informações digitais e sua conexão com ataques digitais. Esse repositório descreve os procedimentos padrão, de acordo com as boas práticas, que permitem **gerar evidências** para impulsionar ações de responsabilização, que **reduzem e expõem** **a impunidade com que os ataques digitais são executados**.

## Para quem é esse repositório?

As informações contidas neste repositório são úteis para pessoas com diferentes perfis, incluindo:

* Pontos de contato (pessoas com capacidade técnica básica)
* Responsáveis por incidentes de segurança para a sociedade civil
* Analistas de laboratórios de ameaças
* Coordenadores de laboratórios de ameaças

Espera-se que **qualquer pessoa com conhecimento técnico básico** possa usar os recursos deste repositório para entender os principais aspectos da análise forense consensual, bem como **procedimentos para identificação e aquisição de evidências**.

Os materiais focados em **análise e apresentação de resultados são especialmente úteis para fortalecer as investigações de laboratórios de ameaças emergentes ou estabelecidos**, pois exigem competências técnicas adicionais, bem como o gerenciamento adequado de riscos.

- :material-school:{ .lg .middle } __Tutoriais__

    ---

    === "Definição


        O tutorial tem como objetivo **aprender conhecimentos práticos** por meio do desenvolvimento de uma atividade. Em outras palavras, é **aprender fazendo**.         [Índice de tutoriais](../../tutorials/){ .md-button }

    === "Exemplo"

        Por exemplo, uma aula para aprender a dirigir geralmente é um tutorial, em que o aluno se familiariza com o veículo, os controles e aprende à medida que avança. O objetivo são as habilidades e o conhecimento, e não ir do ponto A ao B.

## Como navegar no repositório de documentação?

De acordo com a estrutura da Diátaxis, devemos primeiro identificar a necessidade para a qual a documentação é consultada, a fim de entender qual seção e qual recurso é o ideal para atender à nossa necessidade. Especificamente:

* Se o objetivo for **adquirir habilidades e conhecimentos**, examine a seção **tutoriais**.
* Se o objetivo for **melhorar o entendimento e a compreensão** de um tópico, verifique a seção **explainers**.   
* Se o objetivo for **aplicar o conhecimento** para concluir uma tarefa específica, verifique a seção **guias de instruções**.
* Se o objetivo for **obter informações descritivas sobre conceitos ou ferramentas**, consulte a seção **referências**.

À medida que o conteúdo for adicionado, os caminhos de aprendizagem serão mapeados para permitir que os leitores adquiram o conhecimento necessário para concluir tarefas em diferentes estágios do processo de investigação forense.

Agradecemos todos os esforços de indivíduos e comunidades para participar, melhorar e contribuir para o desenvolvimento deste recurso, inclusive em ações como **replicar, verificar, melhorar e ampliar os recursos e as metodologias**.
