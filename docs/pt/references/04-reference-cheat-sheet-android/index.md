# Folha de consulta forense no Android

Este recurso é **parte de um repositório de documentação técnica** que visa estabelecer uma base de conhecimento testada, flexível e acessível para **impulsionar a análise forense consensual em benefício da sociedade civil**. A [estrutura de documentação técnica Diataxis] (https://diataxis.fr/) é usada para organizar o conteúdo.

Esse recurso específico se enquadra na categoria de **referências** e contém uma folha de resumo sobre as **fases de uma investigação forense consensual em um dispositivo Android, incluindo comandos úteis, links e considerações específicas em cada uma das fases.
