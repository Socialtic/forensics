# Referências

Nesta seção, você encontrará materiais de orientação passo a passo. De acordo com [a estrutura Ditaxis] (https://diataxis.fr), as referências contêm **o conhecimento teórico necessário para executar uma tarefa** de forma confiável e sem erros.

Um exemplo de referência seria um mapa. Ele é um recurso claro, neutro e preciso, que apoia a realização de uma tarefa, neste caso, ir do ponto A ao ponto B.
